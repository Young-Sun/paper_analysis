# 📚 AI Paper Analysis & Archive

논문 분석 및 기록하여 나만의 웹 라이브러리를 구축하는 시스템

## 🚀 사용 방법 (회사에서 논문 추가하기)

1. 이 저장소 상단의 `paper_links.txt` 파일을 클릭합니다.
2. 우측 상단의 **연필 아이콘(Edit this file)**을 누릅니다.
3. 분석하고 싶은 **arXiv 링크** 또는 **PDF 주소**를 맨 아랫줄에 추가합니다.
4. **[Commit changes...]** 초록색 버튼을 눌러 저장합니다.
5. 약 3~5분 뒤, 아래 [내 웹사이트 주소]에서 분석 보고서를 확인하세요!

## 🌐 내 전용 논문 아카이브 웹사이트
> **주소:** `https://Young-Sun.github.io/paper_analysis/`

## 🛠️ 시스템 구조
- **Gemini 1.5 Pro:** 최신 AI 모델을 사용해 논문의 핵심 내용을 심층 분석합니다.
- **PyMuPDF:** PDF에서 텍스트를 추출하고, 주요 Figure와 Table 이미지를 자동으로 스크랩합니다.
- **GitHub Actions:** 링크 추가 시 자동으로 분석 스크립트를 실행하고 웹사이트를 업데이트합니다.
- **MkDocs Material:** 분석된 마크다운 결과를 인쇄에 최적화된 깔끔한 웹페이지로 변환합니다.

## 📂 폴더 구조
- `docs/`: 분석된 마크다운 보고서들이 저장되는 곳입니다. (옵시디언 연동 시 이 폴더를 참고하세요)
- `docs/images/`: 논문에서 추출된 그림 파일들이 보관됩니다.
- `analyze.py`: 논문을 분석하는 메인 파이썬 엔진입니다.
- `paper_links.txt`: 분석할 논문 리스트입니다.
