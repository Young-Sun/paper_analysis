    Qwen3-TTS Technical Report 요약

    이 문서는 Qwen Team에서 발표한 Qwen3-TTS 시리즈에 대한 기술 보고서입니다. Qwen3-TTS는 다국어 지원, 제어 가능성, 견고성, 스트리밍 기능을 갖춘 최첨단 텍스트 음성 변환(Text-to-Speech, TTS) 모델군을 소개합니다.

    주요 특징 및 성능

    1. 최첨단 음성 복제 및 제어:
      • 3초 음성 복제를 지원합니다.
      • 설명 기반 제어(Description-based Control)를 통해 완전히 새로운 목소리를 만들거나 출력 음성에 대한 세밀한 조작이 가능합니다.
    2. 대규모 다국어 지원:
      • 10개 언어에 걸쳐 5백만 시간 이상의 음성 데이터로 훈련되었습니다.
      • 스피커 일관성을 유지하는 다국어 생성이 가능합니다.
    3. 실시간 스트리밍:
      • 듀얼 트랙 LM 아키텍처를 채택하여 실시간 합성을 지원합니다.
      • 최소 97ms (0.6B 모델)의 첫 패킷 지연 시간(First-Packet Latency)을 달성하여 초저지연 스트리밍이 가능합니다.
    4. 독창적인 스피치 토크나이저:
      • Qwen-TTS-Tokenizer-25Hz: 25Hz의 단일 코드북 코덱으로 의미론적 콘텐츠에 중점을 두며, Qwen-Audio와 원활하게 통합되어 스트리밍 파형 재구성을 위해 블록 단위 DiT를 사용합니다.
      • Qwen-TTS-Tokenizer-12Hz: 12.5Hz의 16계층 멀티 코드북 설계를 사용하여 극단적인 비트 전송률 감소와 초저지연 스트리밍을 달성하며, 경량 캐주얼 ConvNet을 통해 파형을 재구성합니다.

    모델 아키텍처 및 훈련

    • 아키텍처: Qwen3 LM 제품군을 활용하며, 텍스트 토큰과 음성 토큰을 채널 축으로 연결하는 듀얼 트랙 표현을 사용하여 실시간 합성을 수행합니다.
      • Qwen3-TTS-25Hz: 단일 레벨 음성 토큰을 사용하며, 청크 단위 DiT 모듈로 파형을 재구성합니다.
      • Qwen3-TTS-12Hz: RVQ 토큰을 사용하며, 제로봇 코드북을 예측한 후 MTP(Multi-Token Prediction) 모듈이 잔여 코드북을 생성하는 계층적 예측 방식을 채택합니다.
    • 훈련 단계:
      • 사전 훈련 (Pre-training): (S1) 일반 다국어 데이터 학습, (S2) 고품질 데이터로 지속적인 사전 훈련(CPT), (S3) 최대 토큰 길이를 32,768로 늘려 장문 처리 능력 향상.
      • 사후 훈련 (Post-training): DPO(Direct Preference Optimization)를 사용하여 인간 선호도에 맞춤, GSPO 및 경량 스피커 파인 튜닝을 통해 안정성과 표현력을 향상시킵니다.

    주요 실험 결과

    • 제로샷 음성 복제: Seed-TTS 벤치마크에서 최저 WER(1.24, test-en)을 달성하며 SOTA를 기록했습니다. 12Hz 변형 모델이 25Hz 모델보다 내용 정확도(WER)에서 우수했습니다.
    • 다국어 음성 생성: 10개 언어 중 6개 언어에서 MiniMax-Speech 및 ElevenLabs를 능가하는 우수한 지능성(낮은 WER)을 보였으며, 모든 10개 평가 언어에서 가장 높은 스피커 유사도 점수를 기록했습니다.
    • 교차 언어 생성: zh-to-ko와 같은 어려운 쌍에서 오류율을 66%까지 감소시키며 CosyVoice 시리즈 대비 뛰어난 교차 언어 일반화 능력을 보여주었습니다.
    • 제어 가능한 음성 생성: InstructTTSEval 벤치마크에서 음성 설계(Voice Design) 시나리오에서 오픈소스 모델 중 SOTA를 달성했으며, 대상 스피커 편집에서 GPT-4o-mini-tts를 크게 능가했습니다.
    • 장문 음성 생성: 10분 이상의 긴 오디오 합성에서 기존 청크 기반 시스템의 경계 아티팩트 없이 일관성을 유지하며, 25Hz 변형 모델이 가장 낮은 WER(long-zh 1.517, long-en 1.225)을 기록했습니다.

    결론

    Qwen3-TTS는 두 가지의 혁신적인 스피치 토크나이저와 듀얼 트랙 구조를 통해 고품질, 다국어, 저지연 스트리밍 합성이라는 다양한 요구 사항을 단일 자동 회귀 프레임워크 내에서 통합적으로 해결했습니다. 연구 및 개발 촉진을 위해 토크나이저와 모델 전체가 Apache 2.0 라이선스로 공개되었습니다.
