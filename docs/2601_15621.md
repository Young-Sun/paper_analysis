### 1. 개요 및 주요 특징

Qwen3-TTS는 **다국어, 제어 가능, 강력한 견고성, 스트리밍** 기능을 갖춘 첨단 텍스트 음성 변환(TTS) 모델 시리즈입니다.

* **음성 복제 (Voice Cloning):** 3초 길이의 오디오만으로도 최첨단 음성 복제가 가능합니다.
* **제어 능력:** 텍스트 설명 기반 제어를 통해 완전히 새로운 목소리를 생성하거나 출력 음성에 대한 세밀한 조작(Voice Design, Voice Editing)이 가능합니다.
* **훈련 데이터:** 10개 언어에 걸쳐 5백만 시간 이상의 방대한 음성 데이터로 훈련되었습니다.
* **실시간 합성:** 실시간 합성을 위해 **듀얼 트랙 LM(Dual-track LM) 아키텍처**를 채택했습니다.
* **오픈소스:** 토크나이저와 모델 모두 Apache 2.0 라이선스로 공개됩니다.

### 2. 핵심 기술: 듀얼 스피치 토크나이저

Qwen3-TTS는 LLM과의 원활한 통합 및 초저지연 스트리밍을 위해 두 가지 유형의 음성 토크나이저를 도입했습니다.

#### 1) Qwen-TTS-Tokenizer-25Hz (의미론적 중심)
* **특징:** 25Hz의 단일 코드북 코덱을 사용하여 **의미론적 콘텐츠**에 중점을 둡니다.
* **재구성:** 블록 기반 DiT(Diffusion Transformer)를 통해 스트리밍 파형 재구성이 가능합니다.
* **단점:** 단일 코드북 설계로 인해 초저지연 애플리케이션에는 덜 적합합니다.

#### 2) Qwen-TTS-Tokenizer-12Hz (초저지연 중심)
* **특징:** 12.5Hz의 **다중 코드북(Multi-codebook, 16개 레이어)** 설계를 채택하여 비트 전송률을 극적으로 줄이고 초저지연 스트리밍을 구현합니다.
* **구조:** 첫 번째 코드북은 의미론적 콘텐츠를, 이후 레이어는 음향 세부 정보를 포착합니다.
* **재구성:** 경량의 인과형 ConvNet을 사용하여 스피커 벡터 추출이나 복잡한 확산 모델 없이 파형을 재구성합니다.
* **지연 시간:** **97ms (0.6B 모델)**의 즉각적인 첫 패킷 방출이 가능합니다.

### 3. 아키텍처 및 훈련

* **아키텍처:** 텍스트 토큰과 음향 토큰을 채널 축으로 연결하는 **듀얼 트랙 표현**을 사용하여 실시간 합성을 수행합니다.
    * **Qwen3-TTS-25Hz:** 단일 수준 토큰을 사용하며, 청크 기반 DiT 모듈로 파형을 재구성합니다.
    * **Qwen3-TTS-12Hz:** RVQ 토큰을 사용하며, MTP(Multi-Token Prediction) 모듈을 통해 계층적 예측을 수행합니다.
* **훈련 단계:**
    * **사전 훈련 (Pre-training):** (S1) 일반 다국어 데이터 훈련, (S2) 고품질 데이터로 연속 사전 훈련(CPT), (S3) 최대 토큰 길이 확장(최대 32,768).
    * **사후 훈련 (Post-training):** DPO(Direct Preference Optimization)를 통한 인간 선호도 정렬, GSPO를 통한 기능 및 안정성 향상, 경량 스피커 미세 조정(SFT).

### 4. 실험 결과 및 성능

Qwen3-TTS는 다양한 벤치마크에서 최첨단 성능을 달성했습니다.

* **효율성 (Streaming Efficiency):** 12Hz 변형 모델은 25Hz 변형 모델보다 훨씬 낮은 첫 패킷 지연 시간(97ms 대 150ms @ 1x 동시성)을 보였습니다.
* **제로샷 음성 합성 (Zero-Shot Speech Generation):** Seed-TTS 벤치마크에서 **WER 1.24**를 달성하며 SOTA를 기록했습니다 (12Hz-1.7B 모델 기준).
* **다국어 음성 합성 (Multilingual Generation):** 10개 언어 중 6개에서 WER이 가장 낮았으며, **모든 10개 언어에서 스피커 유사성 점수(SIM)가 최고**였습니다.
* **교차 언어 합성 (Cross-Lingual Generation):** zh-to-ko 생성에서 기존 CosyVoice3 대비 오류율을 약 **66%** 감소시키며 탁월한 일반화 능력을 입증했습니다.
* **제어 가능 음성 생성 (Controllable Generation):** InstructTTSEval에서 **Voice Design(VD) 부문**에서 오픈소스 모델 중 SOTA를 달성했습니다.
* **장문 음성 합성 (Long Speech Generation):** 10분 이상의 긴 오디오 생성 시에도 WER이 가장 낮아(long-zh: 1.517, long-en: 1.225) 아키텍처의 안정성이 입증되었습니다.
